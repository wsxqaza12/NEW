1b. Forest plot

Then use the forest command to draw the forest plot. The split.screen function is used to split a plotting window into different rectangular sections called “screens”. Figure 3.1 is the forest plot of Nishiumra data.

split.screen(c(1,2))
screen(1)
forest(madad.nishimura, type = "sens", xlab = "Sensitivity", cex = .8, snames = nishimura$study_id)

screen(2)
forest(madad.nishimura, type = "spec", xlab = "Specificity", cex = .8, snames = nishimura$study_id)

close.screen(all.screens = T)

Figure 3.1: The forest plot of Nishiumra data.
