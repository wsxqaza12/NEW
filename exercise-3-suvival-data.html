3.1a. Generate the variables for metagen

Then, we have to create the necessary variables for metagen, including the “log(HR)” and the “se(log(HR))”.

chevalier$loghr <- log(chevalier$hr) # log hazard ratio
chevalier$loglci <- log(chevalier$lci) # lower bond of log hazard ratio
chevalier$loguci <- log(chevalier$uci) # upper bond of log hazard ratio
chevalier$seloghr <- (chevalier$loguci-chevalier$loglci)/3.92 # Standard error of log hazard ratio
chevalier$varloghr<-chevalier$seloghr^2

3.1b. Use metagen to carry out meta analysis

# run meta-analysis
chevalier.gen <- metagen(loghr, seloghr, studlab = study, sm = "HR", data = chevalier)

3.2 Scenario 2 - Reporting median survival time

For Scenario 2, we also use metagen to carry out meta analysis. But we have to transform median survival time to hazard ratio.

First, we read the data.

# Read data - Zhang
zhang <- read.csv("~/Desktop/meta/Lab_day2/Zhang.csv")
print(zhang)
##      study year  m1  m2 totals  lnmsr selnmsr                     group
## 1   Fidias 2009 5.7 2.7    309 -0.747  0.0569          Cytotoxic Agents
## 2 Ciuleanu 2009 4.3 2.6    663 -0.503  0.0388          Cytotoxic Agents
## 3   Miller 2009 4.8 3.7    768 -0.260  0.0361 Molecular-targeted Agents
## 4 Cappuzzo 2010 3.1 2.8    889 -0.102  0.0335 Molecular-targeted Agents
## 5    Perol 2010 2.9 1.9    310 -0.423  0.0568 Molecular-targeted Agents
## 6  Surmont 2010 4.1 2.9    173 -0.346  0.0760 Molecular-targeted Agents

3.2a. Transform median survival time to hazard ratio

Then, we have to transform median survival time to hazard ratio by typing the following command.

zhang$logmsr = log(zhang$m2/zhang$m1) # log hazard ratio
zhang$selogmsr = sqrt(1/zhang$totals) # standard error of log hazard ratio

3.2b. Use metagen to carry out meta analysis

# run meta-analysis
zhang.gen <- metagen(logmsr, selogmsr, studlab = study, sm = "HR", data = zhang)
